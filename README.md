# TaskBoard-Pro

A drag and drop task management app with dark/light mode and priority levels.

## Features
- ✅ Add, Delete, Edit Tasks
- ✅ Drag and Drop between To Do, In Progress, Done
- ✅ Dark / Light Mode Toggle
- ✅ Task Priority: High, Medium, Low
- ✅ Data saves in LocalStorage

## Tech Stack
HTML, CSS, JavaScript
